crx_reloadtextures
A simple FiveM /rt chat command script for players to use to reload their game textures! Whether your textures are buggy, glitchy, or non-existent, use this simple reload textures script to fix your in-game textures! NOTE this script will NOT fix server sided texture issues, this will only fix loading issues for clients when using the /rt command!

# crx_reloadtextures
FiveM /rt chat command script for players to use to refresh their game textures!

# Steps on installing
* Drag and drop crx_reloadtextures into your resource folder
* Add paste the following in your server.cfg
* ensure crx_reloadtextures

# Configuration
```
Config = {}

-- The command players will type in chat (without the /)
-- Example: "rt" becomes /rt, "reloadtextures" becomes /reloadtextures
Config.Command = "rt"

-- The message displayed when the reload starts
Config.StartMessage = "Reloading textures... (Game may freeze briefly)"

-- The message displayed when finished
Config.SuccessMessage = "Textures reloaded successfully!"
```

# chat command #
```
"/rt" or whatever you may have changed it to.
```
* start the server
* ensure the resource has started in your console log
* run "/rt" and boom your done
